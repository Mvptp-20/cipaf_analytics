\# CIPAF Design System



\## 🎨 Colors



\- Primary: #6F2495

\- Accent: #BA1C88

\- Highlight: #F4690B

\- Secondary: #171799

\- Teal: #0C8980

\- Yellow: #FFCC00

\- Dark Green: #004900

\- Magenta: #C9008A



\---



\## 🧠 Usage Rules



\- Titles → #3B092B

\- Background → White (#FFFFFF)

\- Accent highlights → #BA1C88

\- Avoid using more than 5 colors in one chart



\---



\## 📊 Chart Guidelines



\- Use consistent color order across visuals

\- Avoid rainbow charts

\- Highlight key insights using accent color

\- Keep labels readable (dark text on light background)



\---



\## 📋 Tables



\- Header background → #6F2495

\- Header text → White

\- Keep grid clean and readable



\---



\## 🎯 Purpose



This design system ensures all CIPAF reports:

\- look consistent

\- are easy to read

\- feel professional and intentional
